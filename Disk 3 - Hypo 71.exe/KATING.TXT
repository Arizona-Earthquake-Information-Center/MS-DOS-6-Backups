    Format of The Earthquake Catalogue of Kandilli Observatory
                 and Earthquake Research Institute
    
    Column  1   2 digits integer form Day
    Column  4   2 Digits integer form Month
    Column  7   4 Digits integer form Year
    Column 12   2 Digits integer form Hour (origin Time)
    Column 14   2 Digits integer form Minute
    Column 16   3 Digits integer form Second (with tenth of second)
    Column 19   1 Digit               indicates error in date and origin time                                          such as D:Day, H:Hour, M:Minute with                                             following two digits. If this column                                             blank, there is no error estimation.
    Column 20   4 Digits integer form Error in second             
    Column 25   4 Digits integer form Latitude (with portion of degree)
    Column 30   4 Digits integer form Error in Latitude in degree
    Column 35   4 Digits integer form Longitude (same as above)
    Column 40   4 Digits integer form Error in Longitude in degree
    Column 45   1 Digit               Reference for previous information
    Column 47   3 Digits integer form Depth
    Column 51   1 Digit               Reference for Depth
    Column 53   3 Digits              Number of recording stations
    Column 57   2 Digits integer form Ms Surface wave Magnitude
    Column 59   1 Digit               Reference for Ms magnitude
    Column 60   2 Digits integer form Mb body waves magnitude
    Column 62   1 Digit               Reference for Mb magnitude
    Column 66   2 Digits integer form ML Local Magnitude
    Column 68   1 Digit               Reference for ML magnitude
    Column 70   2 Digits              Maximum intensity Io (MSK) scale
    Column 73   1 Digit               Reference for Io
    Column 75   3 Digits              Radius of percebtibility
                                      (V intensity) in km
    Column 79   1 Digit               Reference for R5
